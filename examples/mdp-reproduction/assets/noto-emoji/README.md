# Noto Emoji Assets

This folder contains a minimal subset of SVG assets from Google Noto Emoji:

- `robot.svg`: `emoji_u1f916.svg`
- `coin.svg`: `emoji_u1fa99.svg`

Source: https://github.com/googlefonts/noto-emoji

Noto Emoji uses multiple licenses. The upstream repository states that fonts are
under the SIL Open Font License 1.1, tools and most image resources are under
Apache 2.0, and flag assets have separate constraints. Do not add flag assets to
this repository without checking the upstream license details.
